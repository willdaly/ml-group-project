# NSL-KDD Local Dataset Copy

This folder contains the two files used by the project pipeline:

- `KDDTrain+.txt`
- `KDDTest+.txt`

The files are included so the project can be graded and demonstrated without internet access or Kaggle credentials.

## Original Source

- Kaggle dataset mirror: <https://www.kaggle.com/datasets/hassan06/nslkdd>
- Reference paper: Tavallaee, M., Bagheri, E., Lu, W., & Ghorbani, A. A. (2009). *A detailed analysis of the KDD CUP 99 data set*. 2009 IEEE Symposium on Computational Intelligence for Security and Defense Applications. <https://doi.org/10.1109/CISDA.2009.5356528>

## Usage

Run the project with:

```bash
python main.py --data-dir data/nsl-kdd --output-dir outputs
```
